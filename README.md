# RandomQuotePlugin

## 插件介绍

RandomQuotePlugin 是一款可在主页展示随机一言的插件，支持定时自动刷新语句，为页面增添文案氛围。

代码由claude和豆包编写

## 核心功能

1. 在主页展示随机一言文案

2. 每隔**10秒**自动刷新语句

## 数据源说明

本插件使用**自制语句源**，并使用了部分来自 `hitokoto.cn` 提供的语句库，特此感谢！

## 语句新增入口

如需增加语句，可访问：[https://app-aawnhgk0zk01.appmiaoda.com/](https://app-aawnhgk0zk01.appmiaoda.com/)

## 调用接口

```Plain Text

https://backend.appmiaoda.com/projects/supabase291883360291172352/functions/v1/get-random-quote?max_length=45
```
